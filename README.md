📊 Manufacturing Downtime Dashboard (Tableau)
# 📌 Overview

This project presents a Tableau dashboard that analyzes machine downtime across different factories and device types. The goal is to identify where the most downtime occurs and help improve operational efficiency.

# 📷 Dashboard Preview

The dashboard includes:

Downtime Per Factory – compares total downtime across different factory locations
Downtime Per Machine – shows which machines contribute most to downtime
# 🛠️ Tools Used
Tableau (for data visualization)
Dataset containing factory and machine downtime records
# 📊 Key Insights
Some factories have significantly higher downtime compared to others
Machines like Laser Welder and Laser Cutter show the highest downtime
A few machines contribute very little downtime, indicating better efficiency
# 🎯 Purpose

This dashboard helps:

Identify high-risk factories
Detect problematic machines
Support decision-making for maintenance and improvements
# 🚀 How to Use
Open the Tableau dashboard file
Explore factory-wise and machine-wise downtime
Use filters (if available) to drill down into specific data
